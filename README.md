# RPG_Base
RPG_Base is a plugin made for minecraft 1.19.4. It is based around leveling system and skill points which just like in RPG games increase stats.

# Features:
-Skill point leveling system <br>
-Leveling system <br>
-Custom mobs and items <br>
-Custom mining
